# BugMark Slides

BugMark Presentations

| Get Started         | Command                                                  |
|---------------------|----------------------------------------------------------|
| Clone this repo     | `git clone http://github.com/mvscorg/bugmark-slides.git` |
| Install `reveal-md` | `npm install -g reveal-md`                               |
| Author Slides       | Edit files in `src/<file>.md`                            |
| Build Presentations | `bin/build`                                              |
